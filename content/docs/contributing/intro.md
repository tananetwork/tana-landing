Thank you for your interest in contributing to Tana! This section contains documentation for developers who want to contribute to the project.

## For Users vs Contributors

**If you're a user** looking to use Tana:
- See [Quick Start](/guides/quickstart/) for installation
- See [CLI Reference](/tana-cli/intro/) for commands
- See [API Reference](/tana-api/intro/) for API endpoints

**If you're a contributor** looking to develop Tana:
- Continue reading this section
- See [Development Setup](/contributing/setup/) to build from source
- See [System Architecture](/contributing/architecture/) to understand the codebase

## Ways to Contribute

### Code Contributions
- Fix bugs
- Add features
- Improve performance
- Write tests

### Documentation
- Improve existing docs
- Add examples
- Fix typos
- Translate documentation

### Community
- Help others in discussions
- Report bugs
- Suggest features
- Write blog posts or tutorials

## Getting Help

- **GitHub Issues**: Report bugs or request features
- **GitHub Discussions**: Ask questions and share ideas
- **Discord**: Join the community (coming soon)

## Code of Conduct

Be respectful, inclusive, and professional. We're all here to learn and build together.
