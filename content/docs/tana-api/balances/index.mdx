---
title: /balances
description: Manage account balances and currencies
---

Manage balances for users and teams across multiple currencies. All balance amounts are stored as strings to preserve decimal precision.

<details>
 <summary><code className="text-blue-400">GET</code> <code><b>/balances</b></code> <code>(get all balances or specific balance)</code></summary>

### Request

**Get all balances:**
```bash
curl http://localhost:8080/balances
```

**Get specific balance:**
```bash
curl "http://localhost:8080/balances?ownerId=usr_1a2b3c4d5e6f&ownerType=user&currencyCode=USD"
```

### Parameters

<table>
 <tr>
    <td>_parameter_</td>
    <td>_type_</td>
    <td>_necessity_</td>
  </tr>
 <tr>
    <td>**ownerId**</td>
    <td>string (UUID)</td>
    <td>optional (query param)</td>
  </tr>
 <tr>
    <td>**ownerType**</td>
    <td>string (user | team)</td>
    <td>optional (query param)</td>
  </tr>
 <tr>
    <td>**currencyCode**</td>
    <td>string</td>
    <td>optional (query param)</td>
  </tr>
</table>

**Note:** If any query parameter is provided, all three (ownerId, ownerType, currencyCode) are required.

 <summary><code className="text-green-400">POST</code> <code><b>/balances/currencies</b></code> <code>(create currency)</code></summary>

### Request

```bash
curl -X POST http://localhost:8080/balances/currencies \
  -H "Content-Type: application/json" \
  -d '{
    "code": "BTC",
    "type": "crypto",
    "decimals": 8,
    "name": "Bitcoin",
    "symbol": "₿",
    "verified": false
  }'
```

### Parameters

<table>
 <tr>
    <td>_parameter_</td>
    <td>_type_</td>
    <td>_necessity_</td>
  </tr>
 <tr>
    <td>**code**</td>
    <td>string (uppercase)</td>
    <td>required</td>
  </tr>
 <tr>
    <td>**type**</td>
    <td>string (fiat | crypto)</td>
    <td>required</td>
  </tr>
 <tr>
    <td>**decimals**</td>
    <td>number (0-18)</td>
    <td>required</td>
  </tr>
 <tr>
    <td>**name**</td>
    <td>string</td>
    <td>optional</td>
  </tr>
 <tr>
    <td>**symbol**</td>

    <td>optional</td>
