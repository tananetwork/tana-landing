---
title: /blocks
description: Query blockchain blocks
---

Query blocks on the Tana blockchain. Blocks are produced every 6 seconds and contain confirmed transactions.

<details>
 <summary><code className="text-blue-400">GET</code> <code><b>/blocks</b></code> <code>(list blocks)</code></summary>

### Request

```bash
curl "http://localhost:8080/blocks?limit=10&offset=0"
```

### Quick Reference

**List all blocks:**

```bash
curl "http://localhost:8080/blocks?limit=10"
```

```json
[
  {
    "id": "blk_xyz789",
    "height": 42,
    "hash": "0x1a2b3c...",
    "timestamp": "2024-11-08T12:00:06.000Z",
    "transactionCount": 5
  }
]
```

**Get latest block:**

```bash
curl "http://localhost:8080/blocks/latest"
```

```json
{
  "id": "blk_xyz789",
  "height": 42,
  "hash": "0x1a2b3c...",
  "transactionCount": 5,
  "transactions": [...]
}
```

**Get block by height:**

```bash
curl "http://localhost:8080/blocks/42"
```

**Get block by hash:**

```bash
curl "http://localhost:8080/blocks/hash/0x1a2b3c..."
```

### Method Not Allowed

The following methods are not supported:

<code className="text-green-400">POST</code> <code className="text-orange-400">PUT</code> <code className="text-red-400">DELETE</code> <code className="text-purple-400">PATCH</code>
