---
title: /transactions
description: Manage blockchain transactions
---

Create and query transactions on the blockchain. Transactions remain in a pending state until included in a block.

<details>
 <summary><code className="text-blue-400">GET</code> <code><b>/transactions</b></code> <code>(get all transactions)</code></summary>

### Request

```bash

```

<details>
 <summary><code className="text-green-400">POST</code> <code><b>/transactions</b></code> <code>(create transaction)</code></summary>

### Request

```bash
curl -X POST http://localhost:8080/transactions \
  -H "Content-Type: application/json" \
  -d '{
    "from": "usr_abc123",
    "to": "usr_def456",
    "amount": "100.00",
    "currencyCode": "USD",
    "type": "transfer",
    "signature": "sig_..."
  }'
```

### Parameters

<table>
 <tr>
    <td>_parameter_</td>
    <td>_type_</td>
    <td>_necessity_</td>
  </tr>
 <tr>
    <td>**from**</td>
    <td>string (UUID)</td>
    <td>required</td>
  </tr>
 <tr>
    <td>**to**</td>
    <td>string (UUID)</td>
    <td>required</td>
  </tr>
 <tr>
    <td>**amount**</td>
    <td>string (decimal)</td>
    <td>optional*</td>
  </tr>
 <tr>
    <td>**currencyCode**</td>
    <td>string</td>
    <td>optional*</td>
  </tr>
 <tr>
    <td>**type**</td>
    <td>string</td>
    <td>required</td>
  </tr>
 <tr>
    <td>**signature**</td>
    <td>string</td>
    <td>required</td>
  </tr>
 <tr>
    <td>**contractId**</td>
    <td>string (UUID)</td>
    <td>optional</td>
  </tr>
 <tr>
    <td>**contractInput**</td>
    <td>object</td>
    <td>optional</td>
  </tr>
</table>

**Transaction types:** `transfer`, `deposit`, `withdraw`, `contract_call`, `user_creation`, `contract_deployment`

*Required for transfer/deposit/withdraw types

    <td>body</td>
  </tr>

    <td>application/json</td>
<td>

```json
{
  "id": "tx_1a2b3c4d5e6f",
  "from": "usr_abc123",
  "to": "usr_def456",
  "amount": "100.00",
  "currencyCode": "USD",
  "type": "transfer",
  "status": "pending",
  "signature": "sig_...",
  "createdAt": "2024-11-08T12:00:00.000Z"
}
```

</td>
  </tr>
  <tr>
    <td>400</td>
    <td>application/json</td>


  </tr>
