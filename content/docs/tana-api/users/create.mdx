---
title: /users
description: Create a new user on the blockchain
---

Create a new user account by submitting a user creation transaction. The transaction will be pending until included in a block.

<details>
 <summary><code className="text-green-400">POST</code> <code><b>/users</b></code> <code>(create user)</code></summary>

### Request

```bash
curl -X POST http://localhost:8080/users \
  -H "Content-Type: application/json" \
  -d '{
    "publicKey": "ed25519_abc123...",
    "username": "@alice",
    "displayName": "Alice Johnson",
    "bio": "Software developer"
  }'
```

### Parameters

<table>
 <tr>
    <td>_parameter_</td>
    <td>_type_</td>
    <td>_necessity_</td>
  </tr>
 <tr>
    <td>**publicKey**</td>
    <td>string</td>
    <td>required</td>
  </tr>
 <tr>
    <td>**username**</td>
    <td>string</td>
    <td>required</td>
  </tr>
 <tr>
    <td>**displayName**</td>
    <td>string</td>
    <td>optional</td>
  </tr>
 <tr>
    <td>**bio**</td>
    <td>string</td>
    <td>optional</td>
  </tr>
</table>
