## Creation

### new chain

Create a new blockchain (you become the genesis leader).

```bash
tana new chain <name>
```

**Arguments:**
- `<name>` - Chain name

**Examples:**

```bash
tana new chain my-chain
```

```bash
tana new chain testnet
```

**Output:**
```
Creating new blockchain: my-chain

âœ“ Genesis block created
âœ“ Chain config saved to ~/.config/tana/chains/my-chain.json
âœ“ Set as default chain

```

### new node

Create a new node to join an existing chain.

```bash
tana new node
```

**Options:**
- `--connect <url>` - Chain URL to connect to

**Examples:**
```bash
tana new node --connect https://chain.example.com
```

**Output:**
```

âœ“ Node config saved to ~/.config/tana/nodes/node-xyz.json
```

### new user

Create a new user account locally.

```bash
tana new user <username>
```

**Arguments:**
- `<username>` - Username (must start with @)

**Options:**
- `-n, --name <name>` - Display name
- `-b, --bio <bio>` - User bio

**Examples:**

```bash
tana new user @alice --name "Alice Johnson"
```

```bash
tana new user @bob --name "Bob Smith" --bio "Developer"
```

**Output:**

## Deployment

### deploy user

Deploy user account to blockchain.

```bash
tana deploy user <username>
```

**Arguments:**
- `<username>` - Username to deploy

**Options:**
- `-t, --target <url>` - Target chain URL

**Examples:**

```bash
tana deploy user @alice
```

```bash
tana deploy user @alice --target https://chain.example.com
```

### balance

Check user balance.

```bash
tana balance <user>
```

**Arguments:**
- `<user>` - Username or user ID

**Options:**
- `-c, --currency <code>` - Currency code (default: USD)

**Examples:**

```bash
tana balance @alice
```

```bash
tana balance @alice --currency ETH
```

**Output:**
```
Balances for @alice:

  USD    1,000.50
  ETH    2.5
  BTC    0.01

Total currencies: 3
```

### transfer

Transfer funds between users.

```bash
tana transfer <from> <to> <amount> <currency>
```

**Arguments:**
- `<from>` - Sender username
- `<to>` - Recipient username
- `<amount>` - Amount to transfer
- `<currency>` - Currency code

**Examples:**

```bash
tana transfer @alice @bob 100 USD
```

```bash
tana transfer @alice @bob 0.5 ETH
```

**Output:**
